# project-data-saving-bot-PDSB-

**Welcome to PDSB**
This bot gathers information that is gathert from a **Discord Channel** of a project.
The bot saves the _information_ as **project.db** file after starting the logic document once


**HOW TO USE THIS BOT IN A DISCORD SERVER?**
first, type !start to activate this bot on your server, a list of command will be given such as:

!new_project - yeni bir proje eklemek
!projects - tüm projelerinizi listelemek
!update_projects - proje verilerini güncellemek
!skills - belirli bir projeye beceri eklemek
!delete - bir projeyi silmek

These commands are used to allow bot to save a new project you make or edit/ delete it later on. All of the information that will be given will be stored on the file projects.db and cannot be accessed by other users.
